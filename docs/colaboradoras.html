Colaboradoras

PersephoneCSS fue desarrollado por dos estudiantes de Ingeniería de Sistemas Informáticos de la Universidad de El Salvador — FMO.

🌸 Juliana Mariselda Torres Chavarría

TC23011

Desarrollo de Componentes

Diseñó e implementó los componentes visuales del framework: botones, cards, alertas, badges, modales, navbar, sidebar, tablas, formularios y footer — cada uno con sus variantes de color y estilos (solid, outline, pildora, ghost).

Utilidades

Desarrolló el sistema de clases utilitarias: colores de fondo (bg-*), texto (text-*), bordes (border-*, borde-top-*), spacing (p-*, m-*), display (d-*), flexbox (d-flex, j-cont-*) y tipografía (fw-*, fs-*).

Variables

Definió la paleta de colores semántica del framework — desde $primary (Peach Blossom) hasta $muted (Stone Petal) — y las variables de breakpoints, contenedor, botones y navbar.

Archivos: _variables.scss · _botones.scss · _cards.scss · _alerts.scss · _modal.scss · _navbar.scss · _sidebar.scss · _tablas.scss · _formulario.scss · _footer.scss · _utilities.scss · _components.scss
🌿 Fabiola Alexandra Romero Amaya

RA23033

Arquitectura SASS & Mixins

Diseñó la arquitectura modular del framework con @use y namespaces. Construyó todos los mixins reutilizables: responsive(), flex-centrado(), boton-color(), card-base(), tabla-base(), modal-base(), sidebar-base() y más — incluyendo la lógica de contraste automático con list.index() y color.adjust().

Reset & Base

Implementó el reset global del framework (box-sizing, márgenes, tipografía base, normalización de formularios y enlaces) y la configuración del body con padding-top automático para el navbar fijo.

Layout

Desarrolló el sistema de grid de 12 columnas Mobile First con @for, los contenedores (.container, .container-fluid), filas (.row) y columnas responsivas (col-md-*, col-lg-*) usando @extend y math.div().

Configuración GitHub

Configuró el repositorio público, el flujo de trabajo con ramas, el README.md con la guía de instalación, y el despliegue del sitio de documentación en GitHub Pages.

Archivos: _mixins.scss · _base.scss · _layout.scss · _navbar-mixins.scss · _tablas-mixins.scss · main.scss · README.md
✿ Trabajo conjunto: La integración final, revisión de consistencia visual, pruebas de responsividad y el sitio de documentación fueron desarrollados en conjunto por ambas integrantes.
