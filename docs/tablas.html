Tablas

Tablas responsivas con variantes de tamaño, bordes, rayas y color de cabecera. Envuelve con .table-container para scroll horizontal en móvil.

¿Cómo funciona?

La clase base .table aplica estilos mínimos de espaciado y separadores. A partir de ahí se combinan modificadores de tamaño, borde, zebra y color. Las variantes de color (.table-clara-{color} y .table-oscura-{color}) se generan automáticamente con @each sobre $theme-colors.

Siempre envuelve la tabla en .table-container para garantizar scroll horizontal en pantallas pequeñas y bordes redondeados.

HTML — Estructura base 
<div class="table-container">
  <table class="table">
    <thead>
      <tr>
        <th>Columna 1</th>
        <th>Columna 2</th>
        <th>Columna 3</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Dato 1</td>
        <td>Dato 2</td>
        <td>Dato 3</td>
      </tr>
    </tbody>
  </table>
</div>
Las clases de la tabla se combinan entre sí. Por ejemplo: table table-bordered table-striped table-oscura-primary es una combinación válida y habitual.

Clases disponibles

Resumen de todos los modificadores que se pueden combinar sobre .table.

Clase Tipo Descripción
.table Base Tabla base con espaciado y separadores
.table-container Contenedor Scroll horizontal en móvil, bordes redondeados
.table-sm Tamaño Padding reducido (6px 10px), fuente 12px
.table-lg Tamaño Padding ampliado (16px 20px), fuente 16px
.table-bordered Borde Borde en todas las celdas
.table-clean Borde Sin bordes inferiores en celdas
.table-striped Zebra Filas pares con fondo semitransparente
.table-clara-{color} Color Cabecera en tono claro del color + zebra + hover
.table-oscura-{color} Color Cabecera en tono oscuro del color + zebra + hover

Base — .table

La tabla más simple. Solo incluye espaciado interno y un separador entre filas.

# Nombre Ciudad Estado
1 Fatima Cruz San Salvador Activo
2 Fabiola Romero Santa Ana Pendiente
3 Juliana Chavarria San Miguel Inactivo
HTML 
<div class="table-container">
  <table class="table">...</table>
</div>
