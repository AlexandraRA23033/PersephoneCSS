Layout

Sistema Grid

Sistema de rejilla de 12 columnas construido con Flexbox y SASS. Mobile First: las columnas apilan en móvil y se distribuyen horizontalmente en pantallas más grandes.

¿Cómo funciona?

El grid de PersephoneCSS usa tres elementos clave que siempre van juntos en ese orden:

  • .container — envuelve y centra el contenido
  • .row — crea la fila flex con gutter negativo
  • .col-{n} — define el ancho de cada columna (1–12)

Las columnas generadas con @for calculan su ancho con la fórmula (100% / 12) × n. El gutter entre columnas se controla con padding-left y padding-right en cada columna, compensado con margen negativo en .row.

Las clases .col-{n} aplican en todos los tamaños. Usa .col-md-{n} y .col-lg-{n} para sobreescribir a partir de un breakpoint específico.

.row

.row activa el contexto flex con flex-wrap: wrap y aplica márgenes negativos laterales para compensar el padding interno de las columnas y lograr que los bordes queden alineados con el contenedor.

SCSS - _layout.scss 
.row {
display:flex;
flex-wrap: wrap;
margin-right: -($gutter/2);
margin-left: -($gutter/2);
}
SCSS — Base compartida de columnas 
[class*="col-"] {
  position: relative;
  width: 100%;
  padding-right: $gutter / 2;
  padding-left:  $gutter / 2;
  box-sizing: border-box;
  flex-shrink: 0;
  min-width: 0;
}
 
// Generación automática con @for
@for $i from 1 through 12 {
  .col-#{$i} {
    $width: (100% * $i / 12);
    flex: 0 0 $width;
    max-width: $width;
  }
}

Ejemplo básico — tres columnas iguales

Tres columnas que ocupan 4/12 del ancho cada una (igual a un tercio). En móvil apilan al 100%.

.col-12 .col-md-4
.col-12 .col-md-4
.col-12 .col-md-4
HTML 
<div class="row">
  <div class="col-12 col-md-4">Columna 1</div>
  <div class="col-12 col-md-4">Columna 2</div>
  <div class="col-12 col-md-4">Columna 3</div>
</div>

Combinando los tres breakpoints

Se pueden combinar .col-{n}, .col-md-{n} y .col-lg-{n} en el mismo elemento para un control total del layout en cada tamaño de pantalla.

12 / md-6 / lg-3
12 / md-6 / lg-3
12 / md-6 / lg-3
12 / md-6 / lg-3
HTML 
<div class="row">
  <div class="col-12 col-md-6 col-lg-3">Card 1</div>
  <div class="col-12 col-md-6 col-lg-3">Card 2</div>
  <div class="col-12 col-md-6 col-lg-3">Card 3</div>
  <div class="col-12 col-md-6 col-lg-3">Card 4</div>
</div>
