Introducción

Guía completa de PersephoneCSS: instalación, arquitectura SASS, paleta de colores y componentes.

¿Qué es PersephoneCSS?

V4.0.0 Estable

PersephoneCSS es un framework CSS de código abierto desarrollado con SASS bajo la metodología Mobile First. Inspirado en la estética de las peonias y los tonos naturales, ofrece a los desarrolladores un sistema de rejilla de 12 columnas, una librería de componentes UI lista para usar y un conjunto de clases utilitarias para agilizar la maquetación.

A diferencia de otros frameworks, PersephoneCSS nace con una identidad visual propia: una paleta de colores semántica y orgánica que combina tonos tierra, florales y musgosos, garantizando interfaces armónicas sin necesidad de sobreescribir estilos base.

¿Vienes de Bootstrap? PersephoneCSS sigue la misma convención de clase-base + modificador (ej. btn btn-solid-primary), por lo que la curva de aprendizaje es mínima.

Instalación

Existen dos formas de agregar PersephoneCSS a tu proyecto:

1. Descarga directa

Descarga o clona el repositorio y vincula el archivo CSS compilado directamente en tu HTML. Es la opción más sencilla para comenzar.

HTML — Vinculación del CSS 
<!-- En el <head> de tu documento -->
<link rel="stylesheet" href="../css/main.css"/>

<!-- Antes del cierre de </body> (para modales, navbar y sidebar) -->
<script src="main.js"></script>

2. Compilar desde fuente

Si deseas personalizar variables o extender el framework, necesitas Node.js y SASS instalados en tu entorno.

Bash — Compilación con SASS 
# Clonar el repositorio
git clone https://github.com/AlexandraRA23033/PersephoneCSS.git

# Instalar SASS globalmente
npm install -g sass

# Compilar una vez
sass scss/main.scss css/main.css

# Modo observador (recompila al guardar)
sass --watch scss/main.scss:css/main.css
Asegúrate de incluir main.js si tu página utiliza componentes interactivos como la Navbar responsiva, los Modales o el Sidebar.

Estructura del proyecto

PersephoneCSS sigue una arquitectura SASS modular. Cada capa tiene una responsabilidad clara, lo que facilita el mantenimiento y la extensión del framework.

Paleta de colores

PersephoneCSS utiliza nombres semánticos inspirados en la naturaleza. Cada color tiene una variable SASS y genera automáticamente variantes para botones, alertas, badges, cards, navbar y sidebar.

#4E5748
$secondary — Moss Dark
#6E8F6B
$success — Sage Green
#7FA6A0

$warning — Amber Clay
#D36B6B

#EFEBCE
$light — Cream Petal
#2F332B
$dark — Forest Shadow

$accent — Blush Rose
#B8B2A0
$muted — Stone Dust

La lógica de contraste de texto (claro vs oscuro) se resuelve automáticamente mediante SASS, usando list.index() y color.adjust().

_variables.scss — $theme-colors 
$theme-colors: (
  "primary":   #BB8588,
  "secondary": #4E5748,
  "success":   #6E8F6B,
  "info":      #7FA6A0,
  "warning":   #D1A85C,
  "danger":    #D36B6B,
  "alert":     #E85C5C,
  "light":     #EFEBCE,
  "dark":      #2F332B,
  "accent":    #D8A39D,
  "muted":     #B8B2A0,
);

Inicio rápido

Con menos de 20 líneas de HTML ya puedes tener una página con navbar, sidebar y contenido responsivo funcionando.

HTML — Plantilla mínima 
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8"/>
  <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
  <link rel="stylesheet" href="../css/main.css"/>
</head>
<body>

  <nav class="navbar--primary">
    <span class="nav-brand">Mi Sitio</span>
    <button class="nav-toggle">&#9776;</button>
    <ul class="nav-menu">
      <li><a href="#">Inicio</a></li>
    </ul>
  </nav>

  <div class="sidebar-overlay" id="sidebarOverlay"></div>

  <div class="main-wrapper has-navbar">
    <aside class="sidebar-main sidebar-secondary" id="sidebar">
      <div class="sidebar-header">Menú</div>
      <ul class="sidebar-menu">
        <li><a href="#">Sección 1</a></li>
      </ul>
    </aside>
    <button class="sidebar-toggle-btn" id="sidebarToggle">&#9776;</button>
    <main class="main-content">
      <div class="container">
        <h1>Hola, PersephoneCSS</h1>
        <p>Tu primer proyecto con el framework.</p>
      </div>
    </main>
  </div>

  <script src="main.js"></script>
</body>
</html>
