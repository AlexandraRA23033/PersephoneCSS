Badges

Alertas y badges comparten el mismo archivo _alerts.scss y los mismos mixins. Ambos se generan con @each sobre $theme-colors y se diferencian por los parámetros $pildora y $es-badge.

Badge — .badge.{color}

Etiquetas compactas con font-size: 0.75rem y padding: 0.25em 0.6em. A diferencia de las alertas, los badges usan fondo sólido con contraste automático de texto (oscuro o claro según el color).

Referencia SCSS — _alerts.scss

Alertas y badges viven en el mismo archivo y comparten los mismos dos mixins. La diferencia entre uno y otro la controlan los parámetros $pildora y $es-badge.

_alerts.scss 

@use "../abstracts/mixins" as m;
@use "../abstracts/variables" as v;

// Alertas
.alerta {
  @include m.alertas-badges($pildora: false, $es-badge: false);
  display: flex;
  align-items: flex-start;
  gap: 0.5rem;
  width: 100%;
}

.alerta-pildora {
  @include m.alertas-badges($pildora: true, $es-badge: false);
  display: flex;
  align-items: flex-start;
  gap: 0.5rem;
  width: 100%;
}

// Badges
.badge {
  @include m.alertas-badges($pildora: false, $es-badge: true);
}

.badge-pildora {
  @include m.alertas-badges($pildora: true, $es-badge: true);
}

// Genera colores para los 11 tokens del tema
@each $nombre, $color in v.$theme-colors {
  .#{$nombre} {
    @include m.alerta-badges-color($color);
  }
}

// Botón de cierre (solo alertas)
.alerta {
  position: relative;

  .alerta-close {
    margin-left: auto;
    background: none;
    border: none;
    cursor: pointer;
    font-size: 1rem;
    line-height: 1;
    opacity: 0.6;
    color: inherit;
    padding: 0;

    &:hover { opacity: 1; }
  }
}
@mixin alertas-badges() — _mixins.scss 

@mixin alertas-badges($pildora: false, $es-badge: false) {
  display: inline-block;
  border: 1px solid transparent;
  font-weight: 500;
  line-height: 1;

  @if $pildora {
    border-radius: 50px;   // píldora
  } @else {
    border-radius: 4px;    // base
  }

  @if $es-badge {
    font-size: 0.75rem;
    padding: 0.25em 0.6em;
    vertical-align: middle;
  } @else {
    padding: 0.5em 1em;    // alerta
  }
}

@mixin alerta-badges-color($color) {
  &.alerta {
    background-color: rgba($color, 0.15);
    border-color: rgba($color, 0.3);
    color: $color;
  }

  &.badge {
    background-color: $color;
    border-color: $color;
    color: $texto-final; // oscuro o claro según lista de contraste
  }
}
