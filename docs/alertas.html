Con cierre — .alerta-close

Agrega un <button class="alerta-close"> dentro de la alerta. El JS escucha el clic y elimina el elemento con .remove(). Funciona en .alerta y .alerta-pildora.

Referencia SCSS — _alerts.scss

Alertas y badges viven en el mismo archivo y comparten los mismos dos mixins. La diferencia entre uno y otro la controlan los parámetros $pildora y $es-badge.

_alerts.scss 

@use "../abstracts/mixins" as m;
@use "../abstracts/variables" as v;
 
.alerta {
  @include m.alertas-badges($pildora: false, $es-badge: false);
  display: flex;
  align-items: flex-start;
  gap: 0.5rem;
  width: 100%;
}
 
// Variante píldora
.alerta-pildora {
  @include m.alertas-badges($pildora: true, $es-badge: false);
  display: flex;
  align-items: flex-start;
  gap: 0.5rem;
  width: 100%;
}
 
// Genera .{color}.alerta y .{color}.badge para los 11 colores
@each $nombre, $color in v.$theme-colors {
  .#{$nombre} {
    @include m.alerta-badges-color($color);
  }
}
 
// Botón de cierre
.alerta {
  position: relative;
 
  .alerta-close {
    margin-left: auto;
    background: none;
    border: none;
    cursor: pointer;
    font-size: 1rem;
    opacity: 0.6;
    color: inherit;
    padding: 0;
 
    &:hover { opacity: 1; }
  }
}
@mixin alertas-badges() 

@mixin alertas-badges($pildora: false, $es-badge: false) {
  display: inline-block;
  border: 1px solid transparent;
  font-weight: 500;
  line-height: 1;
 
  @if $pildora {
    border-radius: 50px;
  } @else {
    border-radius: 4px;
  }
 
  @if $es-badge {
    font-size: 0.75rem;
    padding: 0.25em 0.6em;
    vertical-align: middle;
  } @else {
    padding: 0.5em 1em;
  }
}
 
@mixin alerta-badges-color($color) {
  &.alerta {
    background-color: rgba($color, 0.15);
    border-color: rgba($color, 0.3);
    color: $color;
  }
}
Clase Border-radius Fondo Borde
.alerta 4px rgba(color, 0.15) rgba(color, 0.3)
.alerta-pildora 50px rgba(color, 0.15) rgba(color, 0.3)
