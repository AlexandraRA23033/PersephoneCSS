components / forms

Formularios

Inputs, textarea, select, checkboxes y estados de validación. Todos construidos sobre @mixin formulario-base con focus, hover y transición incluidos.

Formulario base — .form

Contenedor centrado con ancho máximo de 520px, fondo derivado de $info aclarado 30%, border-radius: 10px y sombra con @mixin sombra($dark, 0.3). Cada campo vive dentro de un .form-group.

Formulario base 

<div class="form">
  <div class="form-group">
    <label class="label" for="nombre">Nombre</label>
    <input class="input" type="text" id="nombre" placeholder="Tu nombre" />
  </div>
  <div class="form-group">
    <label class="label" for="mensaje">Mensaje</label>
    <textarea class="textarea" id="mensaje" placeholder="..."></textarea>
  </div>
  <button class="btn btn-md btn-primary btn-block">Enviar</button>
</div>

Input — .input

Campo de texto base. Hereda @mixin formulario-base: padding, border, border-radius, fondo claro, transición y estilos de :focus con glow en $primary.

Input 

<input class="input" type="text" placeholder="Escribe algo..." />

<!-- Deshabilitado -->
<input class="input" type="text" placeholder="No editable" disabled />

Tamaños — .input-sm / .input-lg

Dos modificadores de tamaño que ajustan padding y font-size. Se combinan con .input.

sm / base / lg 

<input class="input input-sm" type="text" placeholder="Small" />

<input class="input" type="text" placeholder="Normal" />

<input class="input input-lg" type="text" placeholder="Large" />
Clase Padding Font-size
.input-sm 6px 10px 12px
.input 10px 15px 14px
.input-lg 14px 18px 16px

Textarea — .textarea

Comparte el mismo mixin que .input. Altura mínima de 130px con resize: vertical para que el usuario pueda ampliarla.

Textarea 

<div class="form-group">
  <label class="label">Mensaje</label>
  <textarea class="textarea" placeholder="..."></textarea>
</div>

Select — .select

Mismo mixin base que .input con cursor: pointer adicional.

Select 

<div class="form-group">
  <label class="label">País</label>
  <select class="select">
    <option value="">Selecciona una opción</option>
    <option value="sv">El Salvador</option>
    <option value="gt">Guatemala</option>
  </select>
</div>

Estados de validación

.input-valid e .input-invalid cambian el color del borde usando @mixin formulario-estado(). El mensaje auxiliar usa .form-text con modificadores .success o .error.

Correo válido.
Introduce un correo válido.
El campo es opcional.
Validación 

<!-- Campo válido -->
<div class="form-group">
  <label class="label">Correo</label>
  <input class="input input-valid" type="email" value="correo@ejemplo.com" />
  <span class="form-text success">Correo válido.</span>
</div>

<!-- Campo inválido -->
<div class="form-group">
  <label class="label">Correo</label>
  <input class="input input-invalid" type="email" value="correo-sin-arroba" />
  <span class="form-text error">Introduce un correo válido.</span>
</div>

<!-- Texto de ayuda neutro -->
<span class="form-text">El campo es opcional.</span>

Checkbox y Radio — .input-check

Envuelve cada opción en .input-check. El .input interno recibe accent-color: $primary y un escalado sutil de 1.05 para destacar el control nativo.

Checkbox y Radio 

<!-- Checkbox -->
<label class="input-check">
  <input class="input" type="checkbox" />
  Opción A
</label>

<!-- Radio -->
<label class="input-check">
  <input class="input" type="radio" name="grupo" />
  Opción A
</label>
<label class="input-check">
  <input class="input" type="radio" name="grupo" />
  Opción B
</label>
Nota: El accent-color: $primary colorea el control nativo del navegador (checkbox y radio) con el color primario del tema sin necesidad de CSS personalizado extra.

Referencia SCSS — _formulario.scss

Todos los campos comparten @mixin formulario-base. La validación la maneja @mixin formulario-estado() mediante un parámetro $tipo.

_formulario.scss 

@use '../abstracts/mixins' as *;
@use '../abstracts/variables' as *;
@use "sass:color";

.form {
  width: 100%;
  max-width: 520px;
  margin: 0 auto;
  padding: 20px;
  background-color: color.adjust($info, $lightness: 30%);
  border-radius: 10px;
  @include sombra($dark, 0.3);
}

.form-group {
  margin-bottom: 18px;
  display: flex;
  flex-direction: column;
}

.label {
  margin-bottom: 6px;
  font-size: 13px;
  font-weight: 600;
  color: $dark;
}

// .input, .textarea y .select comparten la misma base
.input,
.textarea,
.select {
  @include formulario-base();

  &:hover {
    border-color: color.adjust($muted, $lightness: -10%);
  }
}

// Tamaños
.input-sm { padding: 6px 10px;  font-size: 12px; }
.input-lg { padding: 14px 18px; font-size: 16px; }

// Textarea
.textarea {
  min-height: 130px;
  resize: vertical;
}

// Select
.select { cursor: pointer; }

// Validación
.input-valid   { @include formulario-estado("success"); }
.input-invalid { @include formulario-estado("error"); }

// Mensajes de ayuda
.form-text {
  font-size: 12px;
  margin-top: 6px;
  color: $muted;

  &.success { color: $success; font-weight: 500; }
  &.error   { color: $alert;   font-weight: 500; }
}

// Checkbox y radio
.input-check {
  display: block;
  align-items: center;
  gap: 8px;
  font-size: 14px;

  .input {
    accent-color: $primary;
    transform: scale(1.05);
  }
}
@mixin formulario-base() — _mixins.scss 

@mixin formulario-base {
  padding: 10px 15px;
  border: 1px solid rgba($dark, 0.2);
  border-radius: 6px;
  font-size: 14px;
  background-color: color.adjust($light, $lightness: 10%);
  color: $text-dark;
  @include transicion;

  &:focus {
    outline: none;
    border-color: $primary;
    box-shadow: 0 0 0 3px rgba($primary, 0.2);
  }

  &::placeholder {
    color: rgba($dark, 0.4);
  }
}

@mixin formulario-estado($tipo: "success") {
  @if $tipo == "success" {
    border-color: $success;
  } @else {
    border-color: $alert;
  }
}
Clase Descripción
.form Contenedor centrado, max-width 520px, fondo y sombra
.form-group Agrupa label + campo, flex column, margen inferior
.label Etiqueta del campo, 13px, font-weight 600
.input Campo de texto base con focus y hover
.input-sm / .input-lg Modificadores de tamaño del input
.textarea Área de texto, min-height 130px, resize vertical
.select Lista desplegable con cursor pointer
.input-valid Borde verde ($success) para campos correctos
.input-invalid Borde rojo ($alert) para campos con error
.form-text Mensaje auxiliar debajo del campo
.form-text.success / .error Variantes de color para el mensaje auxiliar
.input-check Envuelve checkbox y radio con accent-color $primary
