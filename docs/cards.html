Cards

Contenedores flexibles con header, body y footer. Variantes de color, layouts y efectos generados con @mixin card-base y @each sobre $theme-colors.

Card base — .card

Estructura mínima: .card con .card-header, .card-body y .card-footer. Usa display: flex en columna con sombra y border-radius: 8px.

Encabezado

Título de la card

Contenido del cuerpo. Aquí va el texto principal del componente.

Pie de la card

Sin footer — .card-sin-footer

Oculta el .card-footer con display: none. El markup del footer puede permanecer en el HTML; el modificador lo esconde.

Referencia SCSS — _cards.scss

La clase base y los modificadores de layout son independientes del color. Los colores se generan con un único @each sin repetición de código.

_cards.scss 

@use '../abstracts/mixins' as comp;
@use '../abstracts/variables' as *;

// Base
.card {
  @include comp.card-base;
}

// Colores — genera .card-{color} para los 11 tokens
@each $nombre, $valor in $theme-colors {
  .card-#{$nombre} {
    @include comp.card-color($valor);
  }
}

// Layout horizontal
.card-horizontal {
  flex-direction: row;

  .card-img {
    width: 40%;
    height: 100%;
    object-fit: cover;
  }
}

// Ocultar footer
.card-sin-footer {
  .card-footer { display: none; }
}

// Contenido centrado
.card-centrada {
  .card-body {
    @include comp.flex-centrado(center, center, column);
  }
}

// Imagen arriba con separador
.card-img-top {
  .card-img {
    border-bottom: 1px solid rgb($dark, 0.1);
  }
}

// Efecto hover
.card-hover {
  @include comp.transicion;

  &:hover {
    transform: translateY(-5px);
    @include comp.sombra($dark, 0.2);
  }
}

// Tipografía interna
.card-title {
  font-size: 1.25rem;
  margin-bottom: 0.5rem;
  font-weight: bolder;
}
.card-text {
  font-size: 0.95rem;
  margin-bottom: 1rem;
}
.card-link {
  @include comp.transicion;
  color: $info;

  &:hover { text-decoration: underline; }
}
@mixin card-base() — _mixins.scss 

@mixin card-base {
  border-radius: 8px;
  overflow: hidden;
  display: flex;
  flex-direction: column;
  @include sombra($dark, 0.1);
  margin-bottom: 1rem;

  .card-header {
    padding: 1rem;
    border-bottom: 1px solid rgba($dark, 0.1);
    font-weight: bold;
  }

  .card-body {
    padding: 1rem;
    flex: 1 1 auto;
  }

  .card-footer {
    padding: 0.75rem 1rem;
    border-top: 1px solid rgba($dark, 0.1);
  }

  .card-img {
    width: 100%;
    height: auto;
    display: block;
    object-fit: cover;
  }
}

@mixin card-color($color-fondo) {
  // Contraste automático según lista de colores claros
  background-color: $color-fondo;
  color: $color-texto; // $text-light o $text-dark

  .card-header,
  .card-footer {
    background-color: rgba($dark, 0.05);
    color: $color-texto;
  }
}
