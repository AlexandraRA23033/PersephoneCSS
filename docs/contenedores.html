Layout

Contenedores

Los contenedores son el bloque de construcción más básico del layout en PersephoneCSS. Son el elemento que contiene, rellena y alinea el contenido dentro de un dispositivo o viewport dado.

¿Cómo funcionan?

PersephoneCSS incluye dos clases de contenedor:

  • .container — ancho máximo fijo que escala por breakpoint
  • .container-fluid — siempre al 100% del ancho disponible

Ambos aplican padding-left: 15px y padding-right: 15px para mantener un espacio interior mínimo. En móvil los dos se comportan igual: ocupan el 100% del ancho. La diferencia aparece en pantallas medianas y grandes.

Los contenedores no se deben anidar entre sí. Para lograr un fondo a todo el ancho con contenido centrado, aplica las clases de fondo (bg-primary, etc.) directamente a la sección y coloca el .container adentro.

Comportamiento por breakpoint

La tabla muestra el max-width que aplica cada contenedor según el tamaño del viewport.

.container

El contenedor estándar. Su max-width cambia en cada breakpoint para mantener el contenido legible y bien proporcionado en cualquier pantalla.

.container

HTML 
<div class="container">
  <!-- Contenido centrado con max-width por breakpoint -->
</div>
SCSS — _layout.scss 
.container {
  width: 100%;
  margin-right: auto;
  margin-left: auto;
  padding-left: 15px;
  padding-right: 15px;
}

@include responde-a("md") {
  .container { max-width: 720px; }
}

@include responde-a("lg") {
  .container { max-width: 1200px; }
}

.container-fluid

Contenedor de ancho completo que se extiende al 100% del viewport en todos los tamaños de pantalla. Útil para secciones hero, fondos de color y barras de navegación.

.container-fluid

HTML 
<div class="container-fluid">
  <!-- Siempre al 100% del ancho disponible -->
</div>
SCSS — _layout.scss 
.container-fluid {
  width: 100%;
  margin-right: auto;
  margin-left: auto;
  padding-left: 15px;
  padding-right: 15px;
}

Patrón común — fondo + contenido centrado

El patrón más habitual en PersephoneCSS es aplicar el color de fondo a la sección y usar .container adentro para limitar el ancho del texto.

Sección con fondo

.bg-primary.p-2 > .container

HTML — Patrón sección + container 
<section class="bg-primary p-3">
  <div class="container">
    <h1>Título del hero</h1>
    <p>Descripción breve.</p>
    <a href="#" class="btn btn-dark btn-md">Comenzar</a>
  </div>
</section>
