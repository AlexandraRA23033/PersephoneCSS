¿Qué es PersephoneCSS?

Un framework CSS desarrollado desde cero con arquitectura SASS modular y paradigma Mobile First. Incluye sistema grid de 12 columnas, 11 tokens de color, componentes reutilizables y utilidades de espaciado, tipografía y flexbox, sin dependencias externas.

Instalación

No hay paquetes que instalar. Solo descarga o clona el repositorio e incluye main.css y main.js en tu proyecto.

Opción 1 — Clonar 
# Clona el repositorio
git clone https://github.com/AlexandraRA23033/PersephoneCSS.git

# Entra al directorio
cd PersephoneCSS
Opción 2 — Descarga directa

Descarga el ZIP desde GitHub y descomprime en tu proyecto.

Descargar ZIP ↓
Vincula los archivos en tu HTML 
<head>
  <link rel="stylesheet" href="css/main.css"/>
</head>
<body>
  <!-- Tu contenido -->
  <script src="main.js"></script>
</body>
Compilación SASS opcional: Instala SASS y compila con sass main.scss main.css --watch. Los parciales están organizados en abstracts/, base/ y components/.

Arquitectura SASS

El framework está estructurado en parciales con separación de responsabilidades. main.scss importa todos los módulos con @use y @forward.

Árbol de parciales 
main.scss           ← punto de entrada
├── _variables.scss    ← $theme-colors, tokens
├── _mixins.scss       ← botones, cards, tablas, flex…
├── _base.scss         ← reset, tipografía base
├── _layout.scss       ← container, grid, wrapper
├── _navbar.scss       ← nav, nav-toggle, nav-menu
├── _sidebar.scss      ← sidebar-main, overlay
├── _botones.scss      ← @each → .btn-{color}
├── _cards.scss        ← @each → .card-{color}
├── _alerts.scss       ← @each → .alerta-{color}
├── _tablas.scss       ← table-clara, table-oscura
├── _formulario.scss   ← input, select, textarea
├── _modal.scss        ← modal, is-open
├── _footer.scss       ← main-footer variantes
└── _utilities.scss    ← @for/@each → spacing, color, flex
Modularidad SASS en práctica: Un solo @each sobre $theme-colors genera las clases de todos los componentes sin repetir código.
Ejemplo — _botones.scss 
// Genera .btn-{c} y .btn-outline-{c} para los 11 colores
@each $nombre, $valor in $theme-colors {
  .btn-#{$nombre} {
    @include botones-base;
    @include boton-color($valor);
    @include boton-tamano("movil");
  }
  .btn-outline-#{$nombre} {
    @include botones-base;
    @include boton-color($valor, "outline");
    @include boton-tamano("movil");
  }
}

JavaScript — main.js

El JS es mínimo y sin dependencias. Maneja el toggle del navbar, apertura/cierre de modales por data-modal-target, toggle del sidebar y cierre de alertas.

Toggle Navbar 
// Abre y cierra el nav-menu en móvil
const navToggle = document
  .querySelector(".nav-toggle");
const navMenu = document
  .querySelector(".nav-menu");

navToggle.addEventListener("click", () => {
  navMenu.classList.toggle("is-active");
});
Apertura de modales 
// Abre modal por data-modal-target
const openBtns = document
  .querySelectorAll("[data-modal-target]");

openBtns.forEach((btn) => {
  btn.addEventListener("click", () => {
    const id = btn.getAttribute(
      "data-modal-target");
    document.getElementById(id)
      .classList.add("is-open");
  });
});
Cierre de alertas 
// Elimina la alerta al hacer click en .alerta-close
document.addEventListener("click", (e) => {
  if (e.target.classList.contains("alerta-close")) {
    const alerta = e.target
      .closest(".alerta, .alerta-pildora");
    if (alerta) alerta.remove();
  }
});
