Botones

Variantes de color, estilos solid y outline, tamaños y estados. Generados con @each sobre el mapa $theme-colors.

Colores del tema

Los botones heredan los 11 colores definidos en $theme-colors.

primary secondary success info warning danger alert light dark accent muted
_variables.scss 
$theme-colors: (
  "primary":   #BB8588,
  "secondary": #4E5748,
  "success":   #6E8F6B,
  "info":      #7FA6A0,
  "warning":   #D1A85C,
  "danger":    #D36B6B,
  "alert":     #E85C5C,
  "light":     #EFEBCE,
  "dark":      #2F332B,
  "accent":    #D8A39D,
  "muted":     #B8B2A0,
);

Tamaños

Tres tamaños definidos por @mixin boton-tamano(). Base mobile-first: btn-sm.

sm, ms, lg 

<button class="btn btn-sm btn-primary">Small</button>

<button class="btn btn-md btn-success">Medium</button>

<button class="btn btn-lg btn-outline-info">Large</button>
Clase Padding Font-size
.btn-sm 8px 16px 14px
.btn-md 12px 22px 16px
.btn-lg 16px 30px 18px

Estados

El estado :disabled aplica opacidad 50% y escala de grises automáticamente.

Estados 

<!-- Botón normal -->
<button class="btn btn-md btn-primary">Normal</button>

<!-- Botón deshabilitado -->
<button class="btn btn-md btn-primary" disabled>Disabled</button>

<!-- Outline deshabilitado -->
<button class="btn btn-md btn-outline-secondary" disabled>
    Disabled
</button>
Nota: Hover y estados activos gestionados por mixins. El hover aplica color.adjust($color, $lightness: -10%) y translateY(-2px).

Bloque

Agrega .btn-block para que el botón ocupe el 100% del ancho del contenedor.

Block 

<button class="btn btn-md btn-primary btn-block">
    Botón ancho completo
</button>
<button class="btn btn-md btn-outline-secondary btn-block">
    Outline ancho completo
</button>

Referencia SCSS — _botones.scss

Todas las clases se generan con un único @each sobre $theme-colors, sin repetición de código.

_botones.scss 

@use '../abstracts/variables' as *;
@use '../abstracts/mixins' as *;

// Genera .btn-{c}, .btn-outline-{c} y .btn-solid-{c}
@each $nombre, $valor in $theme-colors {

  .btn-#{$nombre} {
    @include botones-base;
    @include boton-color($valor);
    @include boton-tamano("movil");
  }

  .btn-outline-#{$nombre} {
    @include botones-base;
    @include boton-color($valor, "outline");
    @include boton-tamano("movil");
  }

}

// Modificadores de tamaño
.btn-sm {
  @include boton-tamano("movil");
}

.btn-md {
  @include boton-tamano("md");
}

.btn-lg {
  @include boton-tamano("lg");
}

.btn-block {
  width: 100%;
}
